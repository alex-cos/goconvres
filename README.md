# goconvres

Convert resource files (PNG, JPEG, binary, etc.) into Go source files containing byte arrays.

## Description

`goconvres` is a command-line tool that reads any input file and generates a Go source file containing a `[]byte` variable with the file's contents. This is useful for embedding static resources directly into Go binaries without using external tools like `go:embed` (available in Go 1.16+) or packing systems.

The generated file includes a package declaration, and a variable holding the byte data formatted with a configurable number of columns for readability. 

To prevent memory exhaustion, `goconvres` enforces a maximum input file size limit of 10 MB.

## Installation

### Prerequisites

- Go 1.23.5 or later

### Install via go install

```bash
go install github.com/alex-cos/goconvres@latest
```

### Build from source

```bash
git clone https://github.com/alex-cos/goconvres.git
cd goconvres
make build   # builds to ./bin/goconvres (or .exe on Windows)
```

## Usage

```bash
goconvres [options] <input file> <output file>
```

### Options

| Option | Alias | Description | Default |
| -------- | ------- | ------------- | --------- |
| `--package` | `-p` | Package name for the generated Go file | `resources` |
| `--name` | `-n` | Variable name for the byte slice | `Resource` |
| `--ncols` | `-c` | Number of bytes per line in the generated array | `12` |
| `--help` | `-h` | Show help | |
| `--version` | `-v` | Print version information | |

### Example

Convert a PNG image to a Go resource:

```bash
goconvres --package images --name logo --ncols 16 input/logo.png output/logo.go
```

Generated `output/logo.go`:

```go
// Code generated by goconvres(1.0.0)
// THIS FILE HAS BEEN AUTOMATICALLY GENERATED, DO NOT EDIT OR MODIFY !
//
// Generated on Thu, 30 Apr 2026 09:23:00

package images

var (
  // logo generated resource from file "logo.png".
  logo = []byte{
    0x89, 0x50, 0x4e, 0x47, 0x0d, 0x0a, 0x1a, 0x0a,
    0x00, 0x00, 0x00, 0x0d, 0x49, 0x48, 0x44, 0x52,
    // ... more bytes ...
  }
)
```

## How It Works

1. Reads the input file as raw bytes.
2. Formats the bytes as hexadecimal values (`0xXX`) separated by commas.
3. Uses a Go template to generate the final source file with:
   - Generation timestamp
   - Package declaration
   - Variable declaration with formatted byte array
